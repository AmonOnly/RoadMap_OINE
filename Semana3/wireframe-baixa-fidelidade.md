# Wireframe de baixa fidelidade (web)

## Pagina 1: Pagina inicial (/)

+--------------------------------------------------------------+
| NAV: [Inicio] [Conteudo] [Quiz]                              |
+--------------------------------------------------------------+
| CABECALHO: titulo, subtitulo, chamada: "Conteudo" "Quiz"     |
| [Cartao config: nome, horas, dias, objetivo, idioma, atualizar]|
+--------------------------------------------------------------+
| PAINEL: XP | Sequencia | Progresso | Fonte do quiz            |
+--------------------------------------------------------------+
| SECAO: escolha da jornada                                    |
| [Botao Conteudo do dia] [Botao Quiz adaptativo]              |
+--------------------------------------------------------------+

## Pagina 2: Conteudo do dia (/content)

+--------------------------------------------------------------+
| NAV: [Inicio] [Conteudo]* [Quiz]                             |
+--------------------------------------------------------------+
| CABECALHO COMPACTO: titulo + subtitulo                       |
| [Config rapida: nome, horas, dias, objetivo, atualizar]      |
+--------------------------------------------------------------+
| PAINEL: XP | Sequencia | Progresso                           |
+--------------------------------------------------------------+
| COLUNA ESQ: Assistente (chat)                                |
| +----------------------------------------------------------+ |
| | bolha IA, input, botao enviar                            | |
| +----------------------------------------------------------+ |
| COLUNA MEIO: Mapa de dias (bolinhas)                         |
| [o o o o o o o] + botao "Marcar dia concluido"               |
| COLUNA DIR: Conteudo detalhado                               |
| - visao geral                                                |
| - conceitos-chave (lista)                                    |
| - formulas (lista)                                           |
| - exemplo guiado                                             |
| - demonstracao                                               |
| - galeria de imagens                                         |
| - recursos externos                                          |
+--------------------------------------------------------------+

## Pagina 3: Quiz (/quiz)

+--------------------------------------------------------------+
| NAV: [Inicio] [Conteudo] [Quiz]*                             |
+--------------------------------------------------------------+
| CABECALHO COMPACTO: titulo + botao iniciar quiz              |
| [Config quiz: horas, idioma, atualizar]                      |
+--------------------------------------------------------------+
| PAINEL: XP | Sequencia | Fonte                               |
+--------------------------------------------------------------+
| COLUNA ESQ: Assistente (chat)                                |
| COLUNA DIR: Questao + alternativas + retorno                 |
+--------------------------------------------------------------+

## Elementos comuns
- Navegacao no topo.
- Cards e botoes grandes para acao rapida.
- Layout responsivo (grid para 1 coluna no mobile).
